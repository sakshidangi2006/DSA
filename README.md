## 🚀 Upcoming Topics

### 🔃 Sorting Algorithms
A deep dive into comparison-based and non-comparison-based sorting techniques.

| Algorithm | Time (Best) | Time (Avg) | Time (Worst) | Space |
|-----------|-------------|------------|--------------|-------|
| Bubble Sort | O(n) | O(n²) | O(n²) | O(1) |
| Selection Sort | O(n²) | O(n²) | O(n²) | O(1) |
| Insertion Sort | O(n) | O(n²) | O(n²) | O(1) |
| Merge Sort | O(n log n) | O(n log n) | O(n log n) | O(n) |
| Quick Sort | O(n log n) | O(n log n) | O(n²) | O(log n) |
| Heap Sort | O(n log n) | O(n log n) | O(n log n) | O(1) |
| Counting Sort | O(n+k) | O(n+k) | O(n+k) | O(k) |
| Radix Sort | O(nk) | O(nk) | O(nk) | O(n+k) |

**Topics to cover:**
- Bubble, Selection, Insertion Sort (brute force basics)
- Merge Sort & Quick Sort (divide and conquer)
- Heap Sort (priority queue based)
- Counting Sort, Radix Sort, Bucket Sort (non-comparison based)
- Custom comparators in C++ (`sort()` with lambda)

### 🔎 Searching Algorithms
Efficient techniques to locate elements in arrays, trees, and answer spaces.

| Algorithm | Time | Space | Requirement |
|-----------|------|-------|-------------|
| Linear Search | O(n) | O(1) | None |
| Binary Search | O(log n) | O(1) | Sorted array |
| Ternary Search | O(log₃ n) | O(1) | Unimodal function |

**Topics to cover:**
- Binary Search (standard, lower/upper bound)
- Binary Search on Answer (finding the minimum/maximum feasible value)
- Search in Rotated Sorted Array
- Find Peak Element
- Search in a 2D Matrix

---

### 🧠 Dynamic Programming
Breaking complex problems into overlapping subproblems with optimal substructure.

**Core Patterns:**
- **1D DP** — Fibonacci, Climbing Stairs, House Robber
- **2D DP** — Unique Paths, Minimum Path Sum, Edit Distance
- **Knapsack** — 0/1 Knapsack, Unbounded Knapsack, Subset Sum
- **LCS / LIS** — Longest Common Subsequence, Longest Increasing Subsequence
- **Palindrome DP** — Longest Palindromic Subsequence / Substring
- **Matrix Chain Multiplication** — Interval DP
- **Bitmasking DP** — Subsets and permutations with bitmask states

**Topics to cover:**
- Memoization vs Tabulation (top-down vs bottom-up)
- Space optimization techniques
- DP on Trees and Graphs
- Digit DP
- DP with Prefix Sums
